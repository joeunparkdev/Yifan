# COEN 346 - Operating Systems
COEN 346 - Programming Assignment 1 - Winter 2021

## Objective
Develop a merge-sort program that gets a list of integers in an input file called input.txt. The program then uses the recursive threading method and a merge sort to sort the list. The ouput will be found in the output.txt file.

The output shall contain:
- the sequence of the strings indicating the starting and stopping of each thread
- the sorted sub-list corresponding to each thread
